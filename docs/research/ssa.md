# Space Situational Awareness

My work in this area focuses on developing **physics-informed orbit determination (PIOD)** methods using angle-only observations, enabling trajectory reconstruction without requiring an initial orbit estimate.

This research also includes:
- Light curve analysis using machine learning to infer space object properties (attitude, shape, material).
- Estimating object size and identifying spacecraft materials from spectroscopic data.
- Collaborations with the *AI4S3* international project on AI for space safety and sustainability.
